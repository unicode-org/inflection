Detailed Description

Provides basic capabilities to Exceptions in Inflection.

This class is meant to be subclassed by Exception. It provides a common interface for throwable exceptions by inheriting specifically from std::exception. It provides two common access methods for data in subclasses - Throwable::what() and Throwable::getMessage(), where what() returns a C-type string (const char*) and getMessage() returns a std::u16string.
